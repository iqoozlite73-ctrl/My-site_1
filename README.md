# Techno News

Веб-проект: простой информационный сайт о технологиях, рынке и поддержке бизнеса.

## Что есть на сайте

- **Главная**: описание проекта и навигация.
- **Business Help**: ресурсы и поддержка для предпринимателей.
- **Market**: обзор курсов валют.
- **Severity**: справочник уровней серьёзности инцидентов.

## Технологии

- HTML5
- CSS3

## Как посмотреть

Сайт доступен через GitHub Pages: [https://iqoozlite73-ctrl.github.io/Techno-news/](https://iqoozlite73-ctrl.github.io/Techno-news/)

## Как запустить локально

1. Клонируй репозиторий: `git clone https://github.com/iqoozlite73-ctrl/Techno-news.git`
2. Открой `index.html` в любом браузере.
